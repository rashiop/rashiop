UdraForge

An upgradeable ERC-1155 game economy with a dynamic crafting system and probabilistic difficulty tiers.

  • Four roles: admin, pauser, upgrader, game admin
  • V1→V2 upgrade adding batch purchasing and per-recipe difficulty, storage layout validated by the OpenZeppelin plugin
  • ReentrancyGuardTransient (EIP-1153), CEI pattern, gas-efficient custom errors
  • Unit, fuzz (×256 runs) and invariant tests with custom handlers

Governance · DAO

UdraDAO

Participation-based governance built on a non-transferable ERC20Votes token, with a live on-chain proposal.

  • UPT governance token with auto-self-delegation, bitmap per-epoch claim tracking and per-epoch mint caps
  • Full OZ Governor + TimelockController stack with quorum and a timelock-owned treasury
  • Snapshot off-chain voting wired to on-chain execution — executed a live proposal on Sepolia
  • Invariant tests (128k random calls) proving supply and balance hold across all transitions
